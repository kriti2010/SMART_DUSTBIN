# QUICK START GUIDE

## Getting Started
1. **Instant Access**: Double-click `smart_dustbin_prototype.html`. It works instantly in any modern browser!
2. **Local Server (Optional)**: If you prefer, run `python -m http.server 8000` in this directory and navigate to `http://localhost:8000/smart_dustbin_prototype.html`.

## Basic Controls
- **Drop Waste**: Simulate a piece of waste being thrown into the bin.
- **Sort Now**: Activates the internal sorting mechanism.
- **Toggle Lid**: Open or close the automatic touchless lid.
- **Sensors**: Visualizes the active infrared/ultrasonic sensors.
- **Flow**: Starts a continuous simulation of waste dropping and sorting.
- **Reset**: Resets the simulation state.

## Viewing Options
Select different camera angles from the top view selector: Isometric, Front, Side, Top, Cutaway, Exploded, or Components.

## Troubleshooting
- **Blank Screen**: Ensure you have an active internet connection on first load (to fetch the Three.js library from CDN).
- **Slow Performance**: Close other demanding browser tabs or applications.
