# AntiGravity Setup

If you are using the AntiGravity platform:

1. Open your workspace.
2. Ensure you have the `smart_dustbin` project folder.
3. To view the prototype, you can start a local server using `python -m http.server 8000` in the directory, or use a Live Server extension if available.
4. The prototype is highly compatible and requires no build steps or `npm install`.
